# Personal Finance Dashboard

A streamlined financial monitoring application designed to provide users with efficient expense tracking and data-driven spending insights.

## Project Overview
This dashboard was engineered to simplify personal expense management. By leveraging Python's data processing capabilities, the application offers an intuitive interface for logging transactions and visualizing fiscal data in real-time.

## Technical Architecture
* **Language:** Python
* **Framework:** Streamlit (Web UI)
* **Data Processing:** Pandas (Data manipulation)
* **Visualization:** Plotly (Interactive graphical analysis)

## Core Functionalities
* **Dynamic Logging:** Real-time capture of financial transactions with automated categorization.
* **Analytical Visualization:** Interactive charts providing granular breakdowns of monthly spending habits.
* **Validation Logic:** Built-in mechanisms to ensure data integrity during input.

## System Requirements & Execution
This project is currently structured for local development.

1. **Clone the repository:**
   `git clone https://github.com/jaswanthr-dev/personal-finance-dashboard.git`

2. **Install dependencies:**
   `pip install -r requirements.txt`

3. **Launch the dashboard:**
   `streamlit run app.py`
